Welcome

This page demonstrates a functional Java EE application running in a container. It can do things like...

  • show environment
  • show running processes
  • kill processes
  • kill itself
  • simulate hanging
  • simulate high load

This simple webapp is based on the Openshift JEE Sample running in a wildfly container built using the wildfly sti template. If you have any suggestions, please email Nick N.
